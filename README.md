# Social Media Chat UI

A modern and responsive **Social Media Chat Interface** built using **HTML** and **Tailwind CSS**. This UI mimics the look and feel of popular chat applications with a sidebar for contacts and a main chat section.

---

## Features

* **Sidebar for contacts** with profile pictures, status, and last message preview.
* **Active chat view** with header showing contact name and online status.
* **Chat messages** with distinct styles for sent and received messages.
* **Timestamps** and read indicators for messages.
* **Message input section** with rounded input field and send button.
* Fully **responsive layout** using Tailwind CSS.
* **Clean design** with shadowed cards and soft color palette.

---

## Technologies Used

* **HTML5** – Structure of the application.
* **Tailwind CSS** – Styling, layout, and responsive design.

## Customization

* Replace profile images in the `<img>` tags with your own images.
* Update contact names and messages in the sidebar or main chat section.
* Tailwind CSS classes can be modified for colors, spacing, and other UI tweaks.
